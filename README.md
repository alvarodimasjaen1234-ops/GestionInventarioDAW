# 🕹️ Gestor de Inventario de Videojuegos

Este es mi primer proyecto de **1º de DAW** (Desarrollo de Aplicaciones Web). Es una aplicación de consola desarrollada en Java para poner en práctica los fundamentos de la programación.

## 🛠️ Tecnologías utilizadas

- **Lenguaje:** Java 17
- **Conceptos:** Programación Orientada a Objetos (POO), Encapsulamiento, Colecciones (ArrayList).
- **IDE:** Visual Studio Code

## 📝 Funcionalidades

- [x] Añadir nuevos videojuegos al sistema.
- [x] Listar todos los juegos guardados.
- [x] Eliminar juegos (Próximamente).

## 🚀 Cómo ejecutarlo

Para probar este proyecto, clona el repositorio y ejecuta:

```bash
javac Main.java
java Main
```
